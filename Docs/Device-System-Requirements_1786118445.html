Tech Requirements Specs

  • CPU: ARMv7 with Neon Support (32-bit), ARM64 or A10+

  • GPU: OpenGL ES 2.0+, OpenGL ES 3.0+, Vulkan

  • Memory: 1GB RAM

  • Disk: 500MB

PC

Mac

  • OS: Windows 7 (SP1+), Windows 10 and Windows 11

  • CPU: x86, x64 architecture with SSE2 instruction set support.

  • GPU: DX10, DX11, DX12 capable.

  • RAM: 1GB

  • OS: High Sierra 10.13+

  • CPU: Apple Silicon, x64 architecture with SSE2.

  • GPU: Metal capable Intel and AMD GPUs

  • RAM: 1GB

Other Requirement specs

  • FPS: 60

    • < 100 Draw Calls at any time

    • 4 players in screen

  • Reference resolution: 750 x 1334

    • 9:16 aspect ratio

    • iPhone 7 / iPhone SE (2nd Generation) default resolution

  • Network requirements: Work seamless on Wi-Fi connection

  • Maximum build app size: < 150 MB

    • Downloadable content with bundles to keep app size in the limit

    • Default build size must contain FTUE content

  • Loading time: < 5sec on High End devices

    • Initial Loading contains 1-2 sec Unity Loading

    • Should include the connection to PlayFab login in a Wifi network

  • Max Amount of Players: 32

  • Max Amount of Dynamic Physics Objects: 600

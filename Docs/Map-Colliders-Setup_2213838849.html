Blast Royale : Map Colliders Setup

Created by Matt Ryan, last modified by Nik Denissov on Jan 04, 2024
  • Props

Props

Collider setup configured by adding n number of QuantumStaticBoxCollider3D components. NOTE: there is no need for Unity BoxCollider objects as a source (they can however be used to pre-calculate the QuantumStaticBoxCollider3D extents then removed). QuantumStaticBoxCollider3D component ensures dynamic objects such as projectiles and characters collide.

Characters

There should be no collider components on an adventure character prefab. TODO: remove Unity capsule collider.

Blast Royale : Device USB tricks

Created by Carlos, last modified on Feb 18, 2025

Logging

Android

For android you need to use adb logcat, you can download from here https://developer.android.com/tools/releases/platform-tools
You can also see inside unity in Window -> Analysis -> Android Logcat

IOS

You can use this https://docs.libimobiledevice.org/libimobiledevice/latest/ amazing project to view the logs

Windows builds are available here: https://github.com/L1ghtmann/libimobiledevice

idevicesyslog -m "BlastRoyale"

Mass install

  • Install Adb (Android debug tools)

    • brew install android-platform-tools

  • Install Libmobiledevice

    • On mac should be brew install libimobiledevice

  • Download the script provided here as an attachment.

  • Move script to a new folder

  • Open a terminal window on that folder

  • Type ./install.sh BUILD_NUMBER and press enter

    • You can install only android or ios with the flags -a and -i (./install.sh -a 4465)

  • Wait for installing on all devices

Caveats

All android devices should have

  • developer mode turned on (Turning this on varies by device, on my samsung I go to software information and click 5 times in kernel version)

  • turn on usb debugging in Developer Options

  • Click trust computer when asked

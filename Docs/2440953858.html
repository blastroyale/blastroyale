Event without bots

  • Set the event field MatchConfig.DisableBots to true

  • Set MatchConfig.MinPlayersToStartMatch to a value that makes sense, this should be a number smaller then the matchmaking configs, because some players may not connect to the room. If you set it to 0 it will not check for min players after matching them together, however this allows exploiting, by creating fake clients and joining matchmaking, so itâ€™s recommended to have a value here.

  • Set OverwriteMatchmaking to:
    If you need check Playfab Matchmaking 

    {
   "QueueName": "battleroyale_1_no_bots", // Playfab matchmaking queue
   "FailsOnTimeout": true, // If we don't find players we don't want to create a solo match for that player
   "QueueTimeoutTimeInSeconds": 175 // How long the client waits before giving up and show players not found
 },

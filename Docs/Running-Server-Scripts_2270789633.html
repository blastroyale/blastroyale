Blast Royale : Running Server Scripts

Created by Gabriel on Dec 15, 2023

Our backend can run scripts locally to perform changes in case we need to do manual operations.

Those scripts are located in the Backend solution in the Scripts project


To run a script you basically need to run the ‚ÄúScripts‚Äù project and it will ask which script you need to run. Works out of the box for staging and dev.

For production, you need to create a playfab_key.txt file in the root of ServerScripts project and add the playfab secret key there.

To get the key you need admin on prod playfab and enter the title settings

And then go to the secret keys tab.

Use the ‚ÄúLocalScriptsKey‚Äù to handle local script operations.

After you create the production key you are ready to run server scripts in production.

Remember to never run a script in production without considering chatting extensively for at least 128 hours in a inquisitor room with tech team to ensure nothing bad is gonna happen.

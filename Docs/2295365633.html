Blast Royale : News System - Posting & Maintaining

Created by Gabriel, last modified by Carlos on Nov 25, 2024

We integrate with playfab news to have a news system where we can post news on the game in real time - which means players will get the news the next time they go to main menu without requiring a re-login.

Players view news on main menu on the top right part of their screen

And when opening they can see the latest 30 news, divided by categories.


Posting News

To post news, you need to developer.playfab.com first, go to the title you want to post the news on, then go Content â†’ News

When creating a new item, you will require to setup the title and contents of the message.
The contents follow a JSON format that reflects our NewsItemData model.

At this stage, we can setup a message, a category (that will be dynamically created if not already), buttons with actions (currently we only support Link action) and images.

An example json:

{
  "text": "This is my first news post",
  "category": "Dev News",
  "image": "https://m.media-amazon.com/images/I/61Y18wJ9pFL._AC_UF894,1000_QL80_.jpg"
  "buttons": [
    {
      "ButtonType": "Hyperlink",
      "ButtonText": "Go to Blast Royale Site",
      "Param": "http://www.blastroyale.com",
      "Color": "green"
    },
    {
      "ButtonType": "Hyperlink",
      "ButtonText": "Go to Google",
      "Param": "http://www.google.com",
      "Color": "yellow"
    }
  ]
}


Having the json with the post contents in hand, you can click the Pencil icon on the playfab page to open the json editor and just put your json there.

Remember to select your new status to Published instead of Unpublished to appear in the game.

And as soon as you post this in playfab, your news will appear in game.

Bear in mind that Rich test is supported, so we can add icons, formatting etc.

Guidelines

We can only have up to 10 positions in News for now, so donâ€™t forget to delete old stuff.

We use three categories:

  • NEWS - we can post general stuff news here, also put surveys etc.

  • EVENTS - specific category where we can put time-sensitive information about activities: streams, tournaments, rewarded campaigns etc., something we want players to be ready at a certain time

  • PATCH NOTES - this one will have only one record about latest (or latest two) patch notes, so players can always look at them inside the game. Maybe post can have two buttons to direct people to the whitepaper for more detailed notes, and the second button will send them into discord

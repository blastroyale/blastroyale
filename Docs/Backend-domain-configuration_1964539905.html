Overview

The domain blastroyale.com is configured in Cloudflare, which provides DDoS protection, DNS management and edge certificates. For access to Cloudflare, please reach out to Miiro Juuso (Unlicensed) , Iikka Hamalainen (Unlicensed) or Matt Ryan (Unlicensed) .

Intermediate domain

In order to provide static environment endpoints for Cloudflare as well as the development team, the domain firstlg.net has been set up in Azure App Services Domains.

This domain is used to generate hostnames automatically with Terraform according to the following schema:

<component>.<environment>.<product>.firstlg.net

for example;

Component: marketplace-api

Environment: prod (production)

Product: br (Blast Royale)

marketplace-api.prod.br.firstlg.net

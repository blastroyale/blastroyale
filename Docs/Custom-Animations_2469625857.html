Blast Royale : Custom Animations

Created by Gabriel, last modified on Sep 05, 2024

Character/Skin/Weapons Animations

We have a character_animation.fbx file in our game.

This file contains all animations of all characters.
This file is in Asset Manager but might require manual replacement so it does not loose asset references.

Updating Meta Files:

  • Download from asset manager the latest version of character_animation.fbx

  • Modify (e.g add animation events)

Updating Animations:

  • Download newest character_animations from asset manager

  • In folders (outside unity) replace the old file with the new file

Adding new Weapon Animations:

We currently use blend tree‚Äôs for differentiating animations.

To add a new animation, it requires a couple of steps:

  • Add new entry in WeaponTypes enum

  • Update the character_animations to include the new animations

  • Open the character animator, located in collections/characters/shared

  • There are 3 blend trees currently: Aim, Run, Attack.

  • You need to open those blend tree‚Äôs in the Animator window and add the new entry for the new WeaponType

  • Then go to weapon configs and change the weapon type of the given weapon so the animations are picked from the blend tree given the weapon type number

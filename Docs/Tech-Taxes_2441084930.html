Blast Royale : Tech Taxes

Created by Gabriel on Aug 01, 2024

Tech needs time, but often, time is more focused into creating a product and tech can be left aside.

To avoid this, we have a concept of tech tax. During development, depending where we touching and what we building, we need to pay tax. Hereâ€™s the current taxes we have:


Dev time numbers are approximate just to give an idea.

Boy Scout Tax: +5% Dev Time

Every-time you touch a code, you should make it a little bit better than it was before. It does not need to be much, but just a little bit, rename a variable, tidy up something.

Tech Debt Tax: +20% Dev Time

Are you touching an area that has tech debt ? We add a little dev tax to either learn, document, or improve the tech debt and avoid making it worse. This is pretty much an addition to the boy scout tax.

Devops Tax: +10% Dev Time

Test Tax: +20% Dev Time

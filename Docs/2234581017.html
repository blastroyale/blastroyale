Blast Royale : Leaderboards (Setup and Management)

Created by Gabriel, last modified by Carlos on Nov 25, 2024

Definition


Leaderboards are the same thing as player statistics. They live in Playfab, whenever you create a leaderboard or a player statistic, the other is created automatically.

Declaration

We define leaderboards in playfab leaderboards (e.g dev leaderboards)

We can easily create new leaderboards there and this will automatically create the metrics.


Seasons


Leaderboard has seasons, they reset every now and then. When creating a leaderboard we have two options to choose how they will reset: Automatic or Manual.

If you click on a leaderboard in the leaderboard pages you can see the seasons, when they reset and the history of seasons too.

Manual Resets


In case your leaderboard has a manual reset setup, you need to obviously manually reset it.
You can do this by clicking on the leaderboard name on the leaderboard page and then clicking ‚ÄúReset Now‚Äù. This will make all metrics of the leaderboard reset to 0 and, if there is automated rewards, they shall be given. This operation can take a few minutes depending how busy PlayFab is.

Setting up Rewards


In the leaderboard page, there‚Äôs a menu on top ‚ÄúPrize Tables‚Äù. If you click there and create a new prize table you can see the rewards configs.

You can setup rewards per range, e.g from 1 to 10 give X amount of things.

There‚Äôs 2 types of rewards:
- Currency like CS or COINS
- Items

Only items that are accepted by the game logic AND they are known by server are able to be rewarded.
Server items can be added without a client update, client items no.

For every reward, you also need to setup the function call ‚ÄúSyncPlayfabInventory‚Äù which will get all items the user has in Playfab and add to his game data. This is important.

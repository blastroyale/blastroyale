Blast Royale : Maintenance/Force Update Config

Created by Carlos, last modified on Nov 25, 2024

Click here if you don't know how to change values

This is used for putting the game into maintenance during release processes and also for forcing players to update the game to newer versions.

Key: GameMaintenanceConfig

{
    "Maintenance": false,
    "MaintenanceMessage": "Game is off go play something else, or touch grass idk",
    "AllowedVersion": "0.24.0",
    "VersionBlockMessage": "Your game version {0} is out of date! Go to the app store to get the latest version {1}"
}

How does it work?

If Maintenance is true, players can't boot the game they will get a popup with the MaintenanceMessage, and they can only close the game. This validation is also made before joining matchmaking or joining custom games.

AllowedVersion is the mechanism to force players to update the game, if they have an older version of the game they will get the VersionBlockMessage when booting or joining games.

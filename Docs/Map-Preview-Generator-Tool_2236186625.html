Blast Royale : Map Preview Generator Tool

Created by Matt Ryan, last modified by Nik Denissov on Jan 04, 2024

Instructions to use the tool to generate a map preview image:

  1. Open the ‚ÄòMap Preview‚Äô window

  1. Configure the following properties
    a) Find ‚ÄúMapPreviewCamera‚Äù prefab in Design/Mapworks and drag this prefab into the scene
    b) export directory where preview images are exported to
    c) MapPreviewSetup prefab reference (MapPreviewCamera from the scene)
    d) render texture reference (find it in the project in Art/RenderTexture/MapPreviewRenderTexture)

  1. Edit URP_Default asset file (located in Art/Settings/RenderPipelineAssets)
    a) Anti-aliasing (MSAA) 8x
    b) Upscaling filter FidelityFx Super resolution 1.0

  1. Open scene that you want to capture preview image for and mark all renderer game objects with tag ‚ÄòMapPreviewObject‚Äô in the hierarchy that you want to render into the capture image. (Note the use of ‚ÄòSelect Children' option right clicking on a given game object in hierarchy for multi selection)

    ‚ÄùWater‚Äù doesn‚Äôt need to be a part of the preview. The boundaries of the preview are determined based on objects that are included as MapPreviewObject

If you want some obstacles to be absent from the preview you have to:

  • disable these obstacles

  • rebake the scene

  • then make a preview

Otherwise if you just disable obstacles then you will see black spots on the ground.

  1. Run the game in the editor and load the desired preview scene, then a) click on the prepare shapshot button b) click on the export snapshot button

A png preview image will be exported <scene-name>.png to your specified export directory

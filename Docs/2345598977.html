Production URL for the tool:

https://mainnet-prod-hub-account-service.blastroyale.com/swagger/index.html

Start using admin tool

To perform any actions with the admin tool you need to be authorized first.

Press Authorize button and authorize with your secret key. Ask Gabriel or Nik how to get your secret personal key.

Set a new wallet for a player

Ask a player for:

  • Game ID (in game) and/or email

  • Old wallet (if possible)

  • New wallet

Then find this player in Playfab to find player‚Äôs ‚ÄúMaster player account ID‚Äù, it will look something like: 515D2AE7D2121CF5

Then we need to first find account of that player and ideally compare their old wallet in a database to the one they provided as a proof.

To do this you need to go to ‚ÄúAuthAdmin -> Get admin/wallet‚Äù and press ‚ÄúTry it out‚Äù

Then paste ‚ÄúMaster player account ID‚Äù to playfabId field and press Execute

Then scroll down and in the response body you will see the wallet that is currently associated with this playfab account

Then to link a different wallet to that account you should use the function AuthWallet -> Post where you specify playfab ID and a new wallet, then do Execute and it sets a new wallet to account

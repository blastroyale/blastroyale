General setup

This page is to help keep track of the current bakery settings used for the best result, both in visual quality and number of lightmaps produced.

The map requires both a directional light and a skylight object.

Easy way to setup is to open a scene where you know the baking was done right, and copy those two objects from that scene into your scene.

Or you can setup things manually:

  1. The skylight should have a bakery skylight component attached.

    1. Current best settings are pure white for the colour, and 1 on the intensity. Everything is default.

  2. The directional light should have a Bakery Direct Light component attached.

    1. Colour should be white

    2. Intensity should be 1

    3. Shadow samples are 24

    4. Indirect intensity is 0

    5. Anti ailiasing is on

Once both these scene objects have been configured, go to the bakery tab > Render Lightmap to bring up the bakery window to start a render.

To access all settings, make sure your settings mode is experimental

Map objects setup

All objects should be marked as Static

There’s an exception for destructible/moving objects, like Wooden gates and Moving gates in the FTUE small arena. They cannot be marked as static, because it prevents the animation from working

All materials should use “FLG/Baked/Static Object” shader

All meshes should have the setup that is on the screenshot below

The only exceptions are:

  • the roofs of enterable buildings

  • the tiles of the ground

  • bushes

“Cast Shadows” should be turned OFF for them. Roofs otherwise will make rooms inside very dark. And for ground tiles it makes no sense to drop any shadow
